Two-part research: (1) Technical feasibility report on multi-AI CLI backend support covering headless modes, system prompts, tool capabilities, models, pricing, and phased implementation roadmap. (2) Legal research on license compatibility, ToS restrictions, and practical risks. Key legal findings: Codex CLI, Gemini CLI, Aider, and Cline are all Apache 2.0 (MIT-compatible). Claude Code is proprietary - cannot be bundled. Anthropic and Google ban OAuth passthrough by third parties. Recommended approach: 'Bring Your Own CLI' model with API key auth only.
