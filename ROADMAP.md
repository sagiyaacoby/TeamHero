# TeamHero Roadmap

## What's Here Now (v3.0)

- Multi-agent orchestration with real-time dashboard
- Full task lifecycle - planning, approval, execution, auto-close
- Agent personalities with persistent short and long memory
- Knowledge base with promotion from task deliverables
- Skills system (Playwright, Remotion, extensible)
- Smart model routing (Haiku/Sonnet/Opus per task type)
- Autopilot and scheduled tasks (recurring and one-time)
- Encrypted secrets vault (AES-256-GCM)
- Parent/child task structure with dependency tracking
- Agent conflict prevention with file scope declarations
- Media library with thumbnail previews
- Round table reviews for team-wide status and recovery
- Interrupted task detection and auto-recovery

## What's Next

These are planned improvements actively being worked toward:

- Skills system expansion - more integrations (Trello, Slack, and others)
- Docker support for easier deployment
- Multi-model routing improvements
- Enhanced agent collaboration patterns
- Improved onboarding and first-run experience
- Community-contributed agent templates
- Better documentation and guides

## Ideas Under Consideration

These are being explored but not yet committed to:

- GitHub Projects integration
- Plugin API for custom skills
- Team configuration import/export
- Webhook support for external triggers
- Task analytics and reporting
- Agent template marketplace
