Your Profile

Team Setup

Name your agent team. This becomes the identity of your orchestration system.

Your Orchestrator

The orchestrator is the brain of your team. It coordinates all agents, manages tasks, and serves as your main point of contact. Give it a name and personality.

★ Next Step: Head to the Command Center and ask your orchestrator to build your team. For example: "Build me a team with a Content Writer and a Researcher"
🔒 Security Note: Agents run with the same file access as your user account. On machines with sensitive data, we recommend using Supervised permission mode - agents will ask for your confirmation before file writes and shell commands. You can change this anytime in Settings. See Help > Security & Secrets for details.
